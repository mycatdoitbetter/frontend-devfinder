## Front-end web Smart DevFinder
<p align="center">

</p>
<p align="center">

  </a>

</p>

To init the Frontend aplication in React, do the follow lines:
```bash
cd web-devfinder
yarn install
yarn start
```
After this, the React will load a the aplication on the page `localhost:3000`
